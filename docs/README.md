# docGenerator API Documentation

Welcome to the docGenerator API documentation. This documentation provides a comprehensive guide to the API endpoints available in the docGenerator application.

## Documentation Structure

The documentation is organized into the following files:

- [API Documentation Overview](api_documentation.md) - Overview of the API, common patterns, and examples
- [Project API Documentation](project_api.md) - Detailed documentation for project-related endpoints
- [Workflow API Documentation](workflow_api.md) - Detailed documentation for workflow-related endpoints
- [Document API Documentation](document_api.md) - Detailed documentation for document-related endpoints

## Getting Started

To get started with the docGenerator API, we recommend:

1. Read the [API Documentation Overview](api_documentation.md) to understand the general structure and patterns of the API
2. Explore the specific endpoint documentation based on your needs:
   - [Project API](project_api.md) for managing projects
   - [Workflow API](workflow_api.md) for managing workflows
   - [Document API](document_api.md) for managing and generating documents

## API Base URL

All API endpoints are prefixed with `/api/v1`.
